# ASIGNACIÓN ALEATORIA Y EQUITATIVA PARA AUXILIARES

Este proyecto fué creado por [Diego Ortiz](https://diego-or.github.io/) por medio de Claude Code

Esta web se hizo pensando en los Jefes de enfermería de la clinica Meintegral de Zipaquirá, la idea es que tengan una forma rápida de asignar turnos sin necesidad de otras aplicaciones externas, optimizando tiempos y aumentando la eficiencia
